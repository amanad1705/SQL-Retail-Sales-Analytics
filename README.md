# SQL Retail Sales Analytics Project

## Project Overview

This project is a comprehensive **SQL Retail Sales Analytics** project built using **PostgreSQL**. The goal of the project is to analyze retail sales data and transform raw transactional data into meaningful business insights that can support better decision-making.

The project follows a complete analytics workflow, starting from **Exploratory Data Analysis (EDA)** and progressing to **Advanced Analytics** techniques such as ranking, cumulative analysis, performance analysis, segmentation, and reporting.

## Project Workflow

The project is divided into two major phases:

### Exploratory Data Analysis (EDA)

* Database Exploration
* Dimensions Exploration
* Date Range Exploration
* Measures Exploration
* Magnitude Analysis
* Ranking Analysis

### Advanced Analytics

* Change-Over-Time Analysis
* Cumulative Analysis
* Performance Analysis
* Part-to-Whole Analysis
* Data Segmentation
* Reporting

## SQL Concepts Used

This project demonstrates the use of several important SQL concepts, including:

* **JOINs** for combining data from multiple tables
* **GROUP BY** for aggregating data
* **ORDER BY** for sorting results
* **Aggregate Functions** such as SUM(), AVG(), COUNT(), MIN(), and MAX()
* **Window Functions** such as RANK(), LAG(), SUM() OVER(), and AVG() OVER()
* **CTEs (Common Table Expressions)** for creating reusable query logic
* **CASE Statements** for conditional logic and segmentation
* **Views** for creating reusable customer and product reports

## Advanced Analytics Features

* **Ranking Analysis:** Identified top and bottom-performing products using RANK() window function.
* **Cumulative Analysis:** Calculated running totals and moving averages to track sales trends over time.
* **Performance Analysis:** Compared current sales with previous year sales using LAG().
* **Customer Reporting:** Created a reusable customer report view with KPIs such as recency, average order value, and average monthly spend.
* **Product Reporting:** Created a reusable product report view with KPIs such as product segment, average order revenue, and average monthly revenue.
